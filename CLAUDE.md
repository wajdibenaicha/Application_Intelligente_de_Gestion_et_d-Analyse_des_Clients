# Application Intelligente de Gestion et d'Analyse des Clients

## Project Overview
Full-stack web app for intelligent customer management, surveys, and AI-based offer recommendations. Supports two roles: Administrateur and Gestionnaire.

## Tech Stack

### Backend
- **Java 21** + **Spring Boot 4.0.3**
- **MySQL** (`base_de_donnee` database on localhost:3306)
- **Maven** (use `./mvnw` wrapper)
- **Spring Data JPA** (Hibernate, ddl-auto=update)
- **Spring Mail** (Gmail SMTP via rayenmonser@gmail.com)
- Server runs on **port 8081**

### Frontend
- **Angular 21.1.0** (standalone components, no NgModules)
- **TypeScript 5.9.2**, strict mode
- **Angular SSR** (Express.js)
- Runs on **port 4200**
- API calls target `http://localhost:8081/api/`

## Project Structure

```
backend/src/main/java/com/example/backend/
├── controller/     # 10 REST controllers
├── models/         # 9 JPA entities
├── service/        # 10 service classes
└── Repository/     # 10 Spring Data JPA repositories

frontend/src/app/
├── login/                    # Auth for Admin & Gestionnaire
├── forgot-password/          # Password reset via email
└── dashboard-gestionnaire/   # Manager dashboard
```

## Key Entities
- **Administrateur** - System admin (full_name, password, email)
- **Gestionnaire** - Manager (linked to Role)
- **Client** - Customer (email, phone)
- **Offre** - Offer/recommendation (AI recommendations + manual offers, status)
- **Questionnaire** - Survey (many-to-many with Questions)
- **Question** / **Reponse** - Survey Q&A
- **Role** / **Permission** - RBAC

## API Routes
Base: `http://localhost:8081/api/`
- `/api/administrateurs` - Admin CRUD + login
- `/api/gestionnaires` - Manager CRUD + login
- `/api/clients` - Client CRUD
- `/api/questionnaires` - Survey management
- `/api/questions` - Question management
- `/api/reponses` - Response management
- `/api/offres` - Offer management
- `/api/roles` - Role management
- `/api/permissions` - Permission management
- `/api/password-reset` - Password reset

## Frontend Routes
- `/login` - Login page (tries Admin first, then Gestionnaire)
- `/forgot-password` - Password recovery
- `/dashboard-gestionnaire` - Manager dashboard
- Default: redirect to `/login`

## Running the Project

### Backend
```bash
cd backend
./mvnw spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm start       # dev server on localhost:4200
npm run build   # production build
```

### Database
- Create MySQL database: `base_de_donnee`
- Schema auto-generated by Hibernate
- Initial data: import `Dump2.sql`

## CORS
Backend allows requests from `http://localhost:4200`. Configured in `BackendApplication.java` and `CorsConfig.java`.
